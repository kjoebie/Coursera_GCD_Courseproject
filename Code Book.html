Code Book

This code book describes the variables for the dataset df_tidy2. This dataset is generated in the script run_analysis.R

UNITS

The units of meausure of the variables are divided by its range (gyroscope, entropy, etc) to normalise it. When units are divided by the same units, the units are cancelled are cancelled out leaving only a ratio.
